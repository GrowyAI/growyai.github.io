## 👶 Growy: 儿童发展时间线 (Beta)

基于脑科学与发展心理学的 **0-18 岁能力培养时间线可视化工具**。

[![License: CC BY-NC 4.0](https://img.shields.io/badge/License-CC%20BY--NC%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc/4.0/)

---

### ✨ 特点

* **实时交互：** 通过滑动条动态查看不同年龄段的关键期。
* **分层架构：** 将能力分为“基石系统”等四大层面。
* **关键期提示：** 明确标识当前年龄段最需关注的能力和培养建议。

---

### 🚀 快速启动

本项目为**单文件 HTML 应用**，无需构建或依赖安装。

1.  **克隆仓库。**
2.  使用现代浏览器**打开 `index.html` 文件**即可。

---

* HTML5 / CSS3
* 原生 JavaScript
